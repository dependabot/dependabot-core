## `dependabot-maven`

Maven support for [`dependabot-core`][core-repo].

### Running locally

1. Start a development shell

  ```
  $ bin/docker-dev-shell maven
  ```

2. Run tests
   ```
   [dependabot-core-dev] ~ $ cd maven && rspec
   ```

[core-repo]: https://github.com/dependabot/dependabot-core
