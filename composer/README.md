## `dependabot-composer`

PHP (Composer) support for [`dependabot-core`][core-repo].

### Running locally

1. Start a development shell

  ```
  $ bin/docker-dev-shell composer
  ```

2. Run tests
   ```
   [dependabot-core-dev] ~ $ cd composer && rspec
   ```

[core-repo]: https://github.com/dependabot/dependabot-core
