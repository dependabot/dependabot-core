## `dependabot-bun`

Bun support for [`dependabot-core`][core-repo].

### Running locally

1. Start a development shell

  ```
  $ bin/docker-dev-shell javascript
  ```

2. Run tests
   ```
   [dependabot-core-dev] ~ $ cd javascript && rspec
   ```

[core-repo]: https://github.com/dependabot/dependabot-core
