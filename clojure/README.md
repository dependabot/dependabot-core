## `dependabot-clojure`

Clojure support for [`dependabot-core`][core-repo].

### Running locally

1. Start a development shell

  ```
  $ bin/docker-dev-shell clojure
  ```

2. Run tests
   ```
   [dependabot-core-dev] ~ $ cd clojure && rspec
   ```

[core-repo]: https://github.com/dependabot/dependabot-core
