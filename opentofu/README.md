## `dependabot-opentofu`

OpenTofu support for [`dependabot-core`][core-repo].

### Running locally

1. Start a development shell

  ```
  $ bin/docker-dev-shell opentofu
  ```

2. Run tests
   ```
   [dependabot-core-dev] ~ $ cd opentofu && rspec
   ```

[core-repo]: https://github.com/dependabot/dependabot-core
