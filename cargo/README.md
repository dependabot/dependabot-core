## `dependabot-cargo`

Rust (Cargo) support for [`dependabot-core`][core-repo].

### Running locally

1. Start a development shell

  ```
  $ bin/docker-dev-shell cargo
  ```

2. Run tests
   ```
   [dependabot-core-dev] ~ $ cd cargo && rspec
   ```

[core-repo]: https://github.com/dependabot/dependabot-core
