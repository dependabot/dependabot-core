## `dependabot-swift`

Swift Package Manager support for [`dependabot-core`][core-repo].

### Running locally

1. Start a development shell

  ```
  $ bin/docker-dev-shell swift
  ```

2. Run tests
   ```
   [dependabot-core-dev] ~ $ cd swift && rspec
   ```

[core-repo]: https://github.com/dependabot/dependabot-core
